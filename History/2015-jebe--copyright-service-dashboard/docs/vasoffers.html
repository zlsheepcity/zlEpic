VAS offer

— инструмент для
а) продажи доступных VAS в догонку к текущему заказу
б) списка недоступных для продажи VAS


Продажа VAS

Сама продажа лежит на совести оператора. Инструмент позволяет оформить продажу в реальном времени при разговоре с клиентом.
  • Список VAS доступных к продаже создается автоматически
  • Оператор может менять цену (включая «бесплатно»)
  • Оператор вручную отмечает (берет ответственность), что клиент согласен оплатить VAS
  • Один или несколько VAS проданных клиенту оформляются в платеж оператором

Список недоступных для продажи VAS

Причины попадания:
  • уже продан
  • клиент не хочет — отмечается вручную оператором
  • VAS не предназначен для этого заказа — вычисляется автоматически
Ни одна из причин не мешает оператору продать недоступный для продажи VAS.
