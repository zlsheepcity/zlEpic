Деление на разделы

Overview
«Home page» для заказа Задачи:
1. Описать текущее состояние заказа вкратце
2. Обратить внимание на важные события или обязательные действия

Order form
Копия формы заполненной клиентом

Задачи:
1. Видеть заказ глазами клиента

Payments
Детали платежей и счетов (не оплаченных платежей) привязанных к заказу

Задачи:
1. Все вопросы с участием платежей и счетов

VAS
Инструмент для поиска новых предложений клиенту Задачи:
1. Найти VAS еще не предложенный клиенту
2. Оформить принятый клиентом VAS

writing process
Вся информация о выполнении заказа Задачи:
1. Показать статус выполнения
2. Показать по шагам всю историю выполнения
3. Ссылка на писателя
4. Хранилище всех файлов привязанных к заказу

event log
Список всех событий связанных с заказом в хронологическом порядке

Ключевые события

  • pre-order events (связанные маркетинговые акции, получение промо-кодов и т.д.)
  • order form submit
  • payment creation (vas etc.)
  • payment confirmation
  • vas offer and client responce
  • looking for writer started
  • writer found
  • writing result

Timeline

  • для незавершенных заказов линия начинается с order form submit и заканчивается сегодняшним днем
  • для завершенных заказов линия начинается с order form submit и заканчивается последним событием в жизни заказа
